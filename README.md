# 🚀 PrepWise – Interview Preparation Platform

## 📌 Overview
PrepWise is a structured interview preparation platform designed to help users practice technical questions effectively.

## 🎯 Features (Phase 1)
- Clean and responsive UI
- Categorized questions (HTML, CSS, JavaScript)
- Light & Dark mode toggle with persistence

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 🌗 Theme
- Ice Blue + Slate (Light & Dark Mode)

## 🚧 Future Scope
- Random question generator
- Answer toggle system
- Score tracking
- Full-stack integration
